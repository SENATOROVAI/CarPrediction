      1.6 ячейка расчет метрик на val датасете
  

2. runs/detect/train28/

      Файлы:
   
      2.1 weights - веса обученной модели
   
      2.2 confusion_matrix.png - матрица ошибок модели
   
      2.3 results.png - графики обучения модели
   
      2.4 results.csv - таблица лоссов и метрик обучения модели на каждой эпохе

4. data/cars_002_learn_3/val/images - val датасет для проверки метрик качества
5. data_in/ - необработанные изображения из парсинга
6. out/result - результаты работы модели с изображениями
7. convert_images.py - преобразование фото в единый jpg формат
8. util.py - функции для инференса модели, используемые в show_1.ipynb
9. Отчет.docx - отчет о проделанной работе
